# Chuck Facts iOS
An iOS App that hits you with a roundhouse kick of Chuck Norris facts.

## Requirements
- iOS 9+
- XCode 10+
- Swift 4+

## Install Guide
- `brew install swiftlint`
- `git clone git@github.com:lecocissoto/chuck-facts-ios.git`
- `pod install`
- U r good to go!

## Dependencies
- RxSwift
